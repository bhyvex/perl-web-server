Pure Perl Web Server

Version 1.0.1

This is an alpha release

What this is

This is an exercise in Perl, to get experience with the raw Sockets and Forking. This is also a brush up on C, as most of the functions in Perl to do these actions are very similar to their C language counterparts.

Features:

The server will create a child process to handle each request. Once a connection is made, the parent will fork and delegate a child process to actually fulfill the request on the connection. Leaving the main server to process connections only.
